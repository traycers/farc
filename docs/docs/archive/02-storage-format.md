# Формат хранилища на диске

```
Тогда текущее состояние:

```
┌──────────────────────────────────────┐
│ Пролог                               │
│ ┌─ Фиксировано навсегда ───────────┐ │
│ │  · магическое число               │ │
│ │  · время каталога (ns)            │ │
│ │  · версия формата                 │ │
│ │  · размер пролога                 │ │
│ │  · кол-во записей в каталоге      │ │
│ │  · размер каталога                │ │
│ ├─ Зависит от версии ──────────────┤ │
│ │  · параметры хранилища ...        │ │
│ └──────────────────────────────────-┘ │
├──────────────────────────────────────┤
│ Каталог (переменная длина)           │
│   для каждого фблока:                │
│     · UUID                           │
│     · состояние / флаги              │
│     · begin_time (ns)                │
│     · end_time (ns)                  │
│     · каналы [64] (uint16, фикс.)    │
│       заполненные слева, остаток = 0 │
│       неиспользованные = 0           │
├──────────────────────────────────────┤
│ CRC32 заголовка (пролог + каталог)   │
├──────────────────────────────────────┤
│ Данные (payload)                     │
├──────────────────────────────────────┤
│ TOC (переменная длина)               │
│   (непрозрачно для Storage)          │
├──────────────────────────────────────┤
│ Эпилог (фиксированный размер)       │
│   · CRC32 данных                     │
│   · CRC32 TOC                        │
│   · размер TOC                       │
└──────────────────────────────────────┘
```

Каждая запись каталога теперь фиксированного размера: UUID(16) + состояние/флаги(1) + begin(8) + end(8) + кол-во каналов(1) + каналы(128) = **162 байта**. При 10 000 фблоках — ~1.6 МБ каталог. Парсинг тривиальный — прыгаешь по записям с постоянным шагом.
```

# вторая версия

Текущее состояние структуры фблока:

```
┌──────────────────────────────────────┐
│ Пролог                               │
│ ┌─ Фиксировано навсегда ───────────┐ │
│ │  · магическое число               │ │
│ │  · время каталога (ns)            │ │
│ │  · версия формата                 │ │
│ │  · размер пролога                 │ │
│ │  · кол-во записей в каталоге      │ │
│ │  · размер каталога                │ │
│ ├─ Зависит от версии ──────────────┤ │
│ │  · параметры хранилища ...        │ │
│ └──────────────────────────────────-┘ │
├──────────────────────────────────────┤
│ Каталог (последовательность TLV)     │
│                                      │
│   Узел = magic + type + size + data  │
│                                      │
│   [magic type=UUID   size data...]   │
│   [magic type=FLAGS  size data...]   │
│   [magic type=BEGIN  size data...]   │
│   [magic type=END    size data...]   │
│   [magic type=CHAN64 size data...]   │
│                                      │
│   Незнакомый type — пропуск по size  │
├──────────────────────────────────────┤
│ CRC32 заголовка (пролог + каталог)   │
├──────────────────────────────────────┤
│ Данные (payload)                     │
├──────────────────────────────────────┤
│ TOC (переменная длина)               │
│   (непрозрачно для Storage)          │
├──────────────────────────────────────┤
│ Эпилог (фиксированный размер)       │
│   · CRC32 данных                     │
│   · CRC32 TOC                        │
│   · размер TOC                       │
└──────────────────────────────────────┘
```


# 3 версия

```
┌─────────────────────────────────────────────────────────┐
│ Пролог                                                  │

│ │  версия формата              2 байта   uint16       │ │
│ │  размер пролога              4 байта   uint32       │ │
│ │  кол-во записей в каталоге   4 байта   uint32       │ │
│ │  размер каталога             4 байта   uint32       │ │
│ │  размер фблока               8 байт    uint64       │ │
│ │                                        итого: 34    │ │
│ ├─ Параметры ─────────────────────────────────────────┤ │
│ │  параметры (JSON, UTF-8)     ? байт                 │ │
│ └────────────────────────────────────────────────────-┘ │
├─────────────────────────────────────────────────────────┤
│ Каталог (последовательность TLV)                        │
│                                                         │
│   Узел = magic(2) + type(2) + size(4) + data(size)      │
│                                заголовок узла: 8        │
│                                                         │
│   UUID    8 + N×16  байт                                │
│   FLAGS   8 + N×1   байт                                │
│   BEGIN   8 + N×8   байт                                │
│   END     8 + N×8   байт                                │
│   CHAN64  8 + N×128  байт                               │
│                                                         │
│   Итого: 40 + N×161 байт                                │
├─────────────────────────────────────────────────────────┤
│ CRC32 заголовка (пролог + каталог)   4 байта  uint32    │
├─────────────────────────────────────────────────────────┤
│ Данные (payload)                                        │
├─────────────────────────────────────────────────────────┤
│ TOC (переменная длина)                                  │
│   (непрозрачно для Storage)                             │
├─────────────────────────────────────────────────────────┤
│ Эпилог                                                  │
│   CRC32 данных                       4 байта  uint32    │
│   CRC32 TOC                          4 байта  uint32    │
│   размер TOC                         4 байта  uint32    │
│                                      итого: 12          │
└─────────────────────────────────────────────────────────┘
```
